# 🎌 Streaming Online de Anime

Uma plataforma web moderna e responsiva para streaming de animes, com uma interface visualmente rica, inspirada nas principais plataformas de entretenimento atuais.

## 🚀 Funcionalidades

-   **Hero Section Dinâmica:** Destaque para os animes principais com banners otimizados e chamadas para ação.
-   **Navegação Inteligente:** Cabeçalho com sistema de busca expansível e menu de navegação rápido.
-   **Cards de Anime:** Sistema de cards interativos com efeitos de hover, exibição de nota (rating), ano de lançamento e gêneros.
-   **Totalmente Responsivo:** Design adaptável para dispositivos móveis, tablets e desktops (Mobile First).
-   **Performance e Acessibilidade:** Uso de unidades relativas (`rem`) para garantir consistência visual em diferentes resoluções e zoom de navegador.
-   **Categorias Organizadas:** Layout em grid para facilitar a exploração de novos conteúdos.

## 🛠️ Tecnologias Utilizadas

-   **HTML5:** Estrutura semântica para melhor SEO e acessibilidade.
-   **CSS3 Moderno:**
    -   Flexbox e Grid Layout.
    -   Variáveis CSS para temas e cores.
    -   Animações e transições suaves.
    -   Unidades `rem` para escalabilidade.
-   **Google Fonts:** Tipografia selecionada (Ubuntu e Roboto) para uma leitura clara e moderna.

## 🎨 Design

O projeto foca em uma paleta de cores escura (Dark Mode), realçando as artes dos animes e proporcionando uma experiência imersiva ao usuário. Os componentes foram desenvolvidos seguindo padrões modernos de UI/UX, com espaçamentos equilibrados e feedback visual em todas as interações.
